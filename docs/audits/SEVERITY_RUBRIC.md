# Platform audit severity rubric

| Level | Definition | Response |
|-------|------------|----------|
| **P0** | Security breach, payment/data loss, legal launch blocker, prod down | Fix before launch; block release |
| **P1** | Broken revenue path, CRM data loss, major UX on primary funnel | Fix in remediation sprint 1 |
| **P2** | Incorrect copy, secondary route, dashboard placeholder, perf Warn | Schedule sprint 2 |
| **P3** | Polish, doc drift, nice-to-have a11y | Backlog |
